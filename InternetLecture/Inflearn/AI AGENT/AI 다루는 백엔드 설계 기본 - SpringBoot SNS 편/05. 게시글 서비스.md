# 38. [AI 다루기] 좋아요, 게시글 조회수
- claude code
  - /context
  - /clear
  - 좋아요 도메인과 API를 만들어줘. 필드는 id, userId, postId로 해줘. 그리고 게시물 좋아요 누를 때 해당 게시물의 좋아요수도 같이 늘게 해줘.
  - 이제 PostViewControler 에서 조회수를 증가시키는 API를 만들어줘. 여기서는 아래처럼 Redis 에 조회수 변경을 기록해줘야해. --- INCREMENT post:view:{postId} SADD post:view:dirty_set postId --- 그리고 매분마다 작동하는 스케쥴러를 만들고, 여기서 SMEMBERS post:view:dirty_set 를 조회하 이 안에 들어있는 postId의 조회수를 업데이트하도록 만들어줘.
  - PostViewService 에서 레디스 조회 부분을 PostViewRepository 로 추상화해서 만들어 리팩토링해줘.
  - view.sh 를 실행해보고 문제를 해결해줘.

# 63. MCP란? 클로드 코드 MCP 활용법
- MCP (Model Context Protocol)
  - AI Model(claude, gpt, ...)이 다른 프로그램들과 소통할 수 있게 해주는 다리 역할의 기술
- ex -> 노션에 논문 리스트
  - 논문 분석 -> 노션 MCP 서버를 통해 논문을 읽고 작성 가능 -> MCP 덕분에 가능하다.
- MCP
  - 다른 프로그램과 소통시 규칙 존재
  - MCP -> AI Model Youtube MCP Server, Notion MCP Server
- Code Hallucination (환각)
  - 존재하지 않는 코드를 제공
  - 최신 버전 분석 -> 하위 버전 구현
  - 하위 버전 사용
- 클로드 서버 
  - context7 MCP Server -> claude code 설치 후 사용
    - 최신 버전 사용하도록 해준다.
  - claude mcp add -transport http <name> <url>
  - [context7](http://github.com/upstash/context7)
    - 추가 코드가 없음
    - claude mcp add --transport http context7 https://mcp.context7.com/mcp --header "CONTEXT7_API_KEY: YOUR_API_KEY" --scope project
      - context7 사용시 API KEY는 설정하지 않아도 된다.
      - 자주 사용시 -> API 발급해서 쓰면 된다.
- mcp 목록 확인
  - /mcp
- 명시적 mcp 사용
  - 문장 use context7
    - use context7 명시

- postgresql
```markdown
# 1. Docker 이미지 pull
docker pull crystaldba/postgres-mcp

# 2. MCP 등록 (운영 환경 - restricted)
claude mcp add --transport stdio postgres -- \
  docker run -i --rm \
  -e DATABASE_URI=postgresql://readonly_user:password@host.docker.internal:5432/your_db \
  crystaldba/postgres-mcp --access-mode=restricted

# 개발 환경은 unrestricted
claude mcp add --transport stdio postgres-dev -- \
docker run -i --rm \
-e DATABASE_URI=postgresql://postgres:123123123@host.docker.internal:5432/실제DB명 \
crystaldba/postgres-mcp --access-mode=unrestricted
```

```markdown
# 운영 DB 연결 전에 읽기 전용 계정 먼저 생성 후 연결
sql-- PostgreSQL
CREATE USER readonly_user WITH PASSWORD 'password';
GRANT CONNECT ON DATABASE your_db TO readonly_user;
GRANT USAGE ON SCHEMA public TO readonly_user;
GRANT SELECT ON ALL TABLES IN SCHEMA public TO readonly_user;

-- MySQL
CREATE USER 'readonly_user'@'%' IDENTIFIED BY 'password';
GRANT SELECT ON your_db.* TO 'readonly_user'@'%';
FLUSH PRIVILEGES;
```

# 77. 커스텀 커맨드: Git 커밋 하기
- 커스텀 명령어 -> 프로젝트로 사용
  - .claude/commands/
    - 이 안에 명령어가 될 이름으로 마크다운 파일 생성
    - commit.md
  - 모든 프로젝트에서 사용할 명령어 경로
    - ~/.claude/commands/
- 네임스페이스 지정
  - .claude/commands/git/commit.md
    - git -> 인터페이스가 될 디렉터리
    - /git:commit

# 78. 🔄 커스텀 커맨드와 스킬 통합 안내
- .claude/commands/git-commit.md
- .claude/skills/git-commit/
  - SKILL.md

둘 다 똑같이 /git-commit 명령어 작동 가능
다만 SKILL 방식은 파일들을 더 체계쩍으로 정리 가능

# 79. 커스텀 커맨드 고급: 동적 파라미터
- $ARGUMENTS -> 커스텀 명령어에 파라미터 입력 가능
  - commit.md -> 커밋 메시지: $ARGUMENTS
  - /git:commit 로그인 기능 완료
    - "로그인 기능 완료" -> $ARGUMENTS 로 입력된다.
  - 모든 인수 매개 변수 -> $ARGUMENTS 
  - $1, $2, ... -> 띄어쓰기 기준으로 매개변수로 들어간다.
- 프론트메터 (Frontmatter)
  - 커스텀 명령어에 대한 메타 데이터 지정 가능
    - 여러 가지 명령어가 있다. 공식 홈 확인
  ```markdown
  ---
  description: git 커밋 생성
  ---
  ```

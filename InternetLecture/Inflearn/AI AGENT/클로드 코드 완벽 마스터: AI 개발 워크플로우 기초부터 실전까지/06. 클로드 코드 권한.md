# 35. 권한 모드 (accept, plan): 프로필 웹 사이트 만들기
- 중요!
- 클로두 코드가 기본적으로 어떻게 행동할 지 결정!
- 4가지 모드 존재
  - default: 표준 동작 - 첫 사용시 권한 요청
  - acceptEdits: 세션에 대해 파일 편집 권한을 자동으로 수락, 다른건 여전히 물어보라는 것 -> 개발 시 편하다! (여러 파일 수정, 생성, ...)
  - plan: 계획 모드 -> 쓰기 작업을 하진 않고, 분석만 해서 계획만 해준다. (중요)
  - interactive: -> 모든 권한 프롬프트를 건너뛴다.
- mode 변경
  - shift tab
  - plan mode on -> @ROADMAP.md 파일을 참고해서 개발해줘.
  - + plan mode -> index.html 에 SNS 링크를 포함하는 섹션을 추가해줘 - github, youtube, instargram, etc 
- 작업해줘를 하기 전 많은 plan mode를 활용한다.
- claude code -> 이미지 첨부
  - cursor는 클립보드에 복사된 이미지가 붙여지는데, 인텔리제이 터미널 안에서는 동작이 안 한다.
  - 이미지 첨부 후 수정 사항 -> plan mode on
    - 수정사항: \ - 웹의 전체적인 컨테이너 폭을 줄여주세요 \ - 안 보이는 이미지는 웹에서 샘플이미지를 사용해서 보이도록 해줘!

# 6. AI 모델 vs AI 코딩 도구: 무엇이 다를까?
- AI 모델 -> 회사마다 철학이 다르다.
  - 어떤 데이터, 중점, 답변 스타일
    - Claude: 신중하게 -> AI에게 원칙을 가르쳐 스스로 판단 -> 코딩 정확도 높다.
    - GPT: 폭넓게 -> 사람들 선호 답변 대량 학습 -> 범용, 유연한 답변에 강함
    - Gemini: 멀티미디어 -> 텍스트, 이미지, 영상 동시 이해 -> 멀티 미디어 이해 특화
  - 생각 O, 행동 X
    - 행동하게 만드는 코딩 도구
- 터미널 도구
  1. 컨텍스트 엔지니어링
     - 과거 -> 어떻게 말할지가 중요
     - 현재 -> AI가 무엇을 알고 있는지가 중
  2. 네이티브 통합
- AI 에이전트 실패 주 원인 -> 컨텍스트 관리
  - 불필요 정보 쌓임 -> 초기 지시 묻힘 -> 방향 상실
  - Claude Code
    - 컨텍스트 엔지니어링 도구 제공 -> 필요한 정보만 정확하게
    - 5가지 무기
      - CLAUDE.md
        - 프로젝트 규칙 미리 전달
      - Skills
        - 필요할 때만 전문 지식 활성화
      - MCP
        - 토큰 46.9% 절감
      - Hooks
        - 상황별 맥락 자동 주입
      - Sub-agents
        - 요약된 결과만 반환 -> 메인 대화 깔끔 유지
    - 네이티브 통합
      - CLAUDE를 만든 회사가 Claude Code를 만들었다.
- AI 시대 개발 본질을 이해하기
  - 모델과 도구를 구분
  - 트렌드를 읽음
  - 컨텍스트의 중요성
  - AI를 개발 파트너로 활용

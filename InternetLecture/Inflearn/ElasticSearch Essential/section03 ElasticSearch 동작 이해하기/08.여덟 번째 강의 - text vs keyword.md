# 여덟 번째 강의 - text vs keyword
- 두 다 문자열을 나타내기 위한 타입이다.
- text 타입
  - 전문 검색 (full-text search)을 위해 토큰이 생성된다.
- keyword 타입
  - 정확한 일치를 위해 토큰이 생성된다.
- i am a boy 검색
  - text: i, am, a, boy
  - keyword: i am a boy
    - boy로 검색하면 안 나온다.
    - 색인 속도가 더 빠르다.
- 문자열 필드가 동적 매핑될 경우 text, keyword 두 타입이 모두 생성된다.
  - 정적 매핑을 통해 성능을 올릴 수 있다.
- 두 타입은 검색 과정의 차이가 있다.
- 사용 예시
  - text
    - 주소, 이름, 물품 상세 정보
  - keyword
    - 성별, 물품 카테고리
